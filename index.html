Ground Plane Segmentation from Unstructured Point Clouds

Tim Fratangelo & Bret Minnehan

Download this project as a .zip file Download this project as a tar.gz file

Description

The aim of this project is generate a the 3D mesh of the ground plane segmented from an unstructured three dimensional point cloud data taken from an aerial vehicle. We plan on a multistage approach consisting of filtering, segmentation and three dimensional mesh generation. Mean filtering will be used to reduce the inherent Gaussian white noise found in LIDAR data sets. Segmentation research will be focused mainly on the graph cuts approach as proposed by Boykov et al.. In the final stage the segmented ground point cloud will be made into a 3D mesh using an algorithm based on the work by Scheidegger et al.
